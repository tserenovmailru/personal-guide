# Персональное руководство

Это твоя личная база для развития. Здесь хранятся твой профиль, фокус на текущий период, практики и планы.

## Что тут есть

- **profile.md** — твой текущий уровень и на чём сосредоточиться
- **worldview.md** — ключевая идея, которая сейчас важна
- **methods.md** — 3 конкретных практики на эту неделю
- **weekly/** — планы на неделю
- **daily/** — планы на каждый день

## Как пользоваться

1. Каждое утро открывай `daily/` и смотри, что запланировано на сегодня
2. В начале недели обновляй `weekly/` — что хочешь сдвинуть
3. Раз в месяц перечитывай `profile.md` — актуален ли фокус?

Все файлы можно редактировать вручную прямо на GitHub или через IWE-ассистента.
