# Моё рабочее пространство IWE

Это ваш личный репозиторий знаний, подключённый к [IWE Knowledge Gateway](https://mcp.aisystant.com).

## Структура

- `inbox/` — входящие заметки, черновики, захваченные идеи
- `docs/` — рабочие документы, стратегии, планы

## Как использовать

Работайте через AI-ассистент (claude.ai, ChatGPT, бот). Все файлы в этом репозитории автоматически индексируются и доступны для поиска через Gateway.

Вы можете:
- **Искать** по своим заметкам: «Найди мои записи про...»
- **Записывать**: «Сохрани эту идею в inbox»
- **Удалять**: «Удали файл inbox/старая-заметка.md»

Подробнее: [IWE — Intellectual Work Environment](https://system-school.ru)
