# Персональное руководство IWE

Это живой документ пилота программы личного развития в IWE (Intellectual Work Environment).

## Структура

| Файл | Назначение |
|------|-----------|
| `profile.md` | RCS-профиль: слоты, baseline, bottleneck |
| `worldview.md` | Мировоззренческая фаза и ключевые сдвиги |
| `methods.md` | Методы и практики под текущий bottleneck |
| `weekly/` | Недельные гипотезы (ISO-формат) |
| `daily/` | Тактика на день |

## Ступень и домен

- **Ступень:** 2 — Практикующий
- **Домен:** generic (уточнить при обновлении профиля)

## Как обновлять

1. После каждой значимой практики — обновить `daily/YYYY-MM-DD.md`
2. В начале каждой недели — создать `weekly/YYYY-Www.md` с гипотезой
3. При смене bottleneck — обновить `profile.md` и `methods.md`
4. Раз в месяц — сверить ступень и пересмотреть `worldview.md`
