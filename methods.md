# Методы и практики

## Минимальный набор для Ступени 2

### 1. Слот-практика

**Что:** Ежедневно выделяй 1 слот (25–45 мин) на целенаправленную работу с одним методом.  
**Как:** Выбери слот (W / M1 / M2 / M4), определи конкретный метод, зафикси рабочий продукт.  
**Зачем:** Развитие идёт через практику с осознанием, а не через накопление информации.

### 2. Мыслительное письмо

**Что:** Писать для мышления, а не для записи.  
**Как:** Открой `daily/YYYY-MM-DD.md`, задай себе вопрос и пиши ответ 10–15 минут без редактирования.  
**Зачем:** Письмо — это экстернализация мышления. Невысказанная мысль — не мысль.

### 3. Системный разбор

**Что:** Разобрать любую ситуацию через системные понятия.  
**Как:** Определи: целевая система → надсистема → создатели → рабочие продукты → роли.  
**Зачем:** Тренирует системное видение на реальном материале.

## Когда обновлять

- При определении bottleneck через Диагноста — добавить специфические методы под слабое место
- При переходе на ступень 3 — добавить методы работы с окружением и стейкхолдерами
